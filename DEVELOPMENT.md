# TestCase Manager for PyCharm

Kotlin version: 2.3.20
Plugin version: 0.1.0
Platform: PyCharm 2023.2.7

## Development Setup

### Prerequisites

- Java 21
- Gradle 9.4+

### Build

```bash
./gradlew build
```

### Run IDE with Plugin

```bash
./gradlew runIde
```

### Test

```bash
./gradlew test
```

## Dependencies

- SnakeYAML 2.2 - YAML parsing library
- Python Plugin - PyCharm Python support
